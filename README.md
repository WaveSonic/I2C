# CPU Temperature Reader (Rust)

Програма для моніторингу стану системи (CPU, RAM, диск, батарея) з використанням WMI/ACPI.

## Встановлення

1. Клонувати репозиторій:
   git clone https://github.com/WaveSonic/I2C.git

2. Перейти в каталог проєкту:
   cd I2C

3. Скомпілювати програму:
   cargo build --release

## Запуск

Програма потребує прав адміністратора для доступу до температурних WMI/ACPI-даних.

Після компіляції запустіть:

target\release\cpu_temp_reader.exe

Windows автоматично запросить підвищення прав (UAC).

## Функціональність

- Температура CPU
- Завантаження CPU
- Частота CPU
- RAM (використання)
- Диски (завантаження)
- Батарея

## Примітка

Температура отримується через ACPI (MSAcpi_ThermalZoneTemperature), тому може відрізнятися від значень у фірмових утилітах виробника.
